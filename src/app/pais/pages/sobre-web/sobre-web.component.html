Sobre la construció

Aquí trobaràs de manera esquemàtica, el que considero més interessant del que he aplicat en aquest exercici.

  • Modularització: Estructura pensada per a mitja escala amb un modul de tipus Shared i un l'altre, per a la resta de continguts
  • Rutes, RouterLink i RouterLinkActive
  • Components per a pàgines, components reutilitzats per al cercador i per la taula. Serveis, interfaces, routing.module...
  • Peticions http
  • Tipat de dades amb quicktype per a la interface del array
  • Us de API keys i filtratge de continguts.
  • Us esporàdic d'alguna pipe
  • DebounceTime per a crear les sugerències del cercador
  • Us de RxJs, amb el switchMap

Espero que a pesar de ser tot encara força bàsic, li donis un cop d'ull. Salutacions!
